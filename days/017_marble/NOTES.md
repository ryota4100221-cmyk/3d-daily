# Day 017 — Week 3 · Marble（フラグメントノイズ / fragment noise gradient）★★★★

## 狙い
画面全体を1枚のフラグメントシェーダで描く（Day 015 系譜、ジオメトリなし）が、
土台のノイズを一段引き上げる。Day 015/016 の依存ゼロ *value*-noise fBm から、
**本物の 2D シンプレックスノイズ**（Gustavson / Ashima）へ。さらに今日の新技法は、
その**解析微分版 `snoised`**——ノイズ値と勾配（∂/∂x, ∂/∂y）を1回の呼び出しで返す変種。
勾配がタダで手に入ると、素のノイズでは描けない「流れる大理石のグラデーション」が作れる。

## 使った技法（今日の新規＝★）
- ★ **解析微分つき 2D シンプレックスノイズ `snoised`**：値＋勾配を同時に返す。
  `value = 130·Σ mᵢ⁴(gᵢ·xᵢ)`、`∇ = 130·Σ[−8 mᵢ³(gᵢ·xᵢ)xᵢ + mᵢ⁴ gᵢ]`（dmᵢ/dv = −2xᵢ より）。
- ★ **導関数 fBm（IQ erosion）**：各オクターブの振幅を走行勾配で `1/(1+|d|²)` 減衰。
  斜面がディテールを抑え、TVノイズでなく浸食された石／大理石の脈になる（＝有機的グラデーション）。
- ★ **勾配シェーディング（emboss）**：蓄積した ∇ を実スロープとして `N=(−∇,1)` にし
  ライト方向とドット。ジオメトリゼロで平面クアッドに起伏の陰影が乗る。
- **IQ ドメインワーピング**：`pattern(p)=fbm(p + 4·fbm(p + 4·fbm(p)))`。自身を2回通して
  ノイズを流れる大理石へ。中間ワープベクトル q, r を色に折り込み、脈が独自のトーンを運ぶ
  （paper → sand → ink → 差し色 terracotta）。
- 各オクターブでドメインを回転行列で回し、脈が格子状にグリッド化しないように。
- ポインタ（Day 015/009 系譜）：カーソルをアスペクト空間へ→damp平滑化し、`exp(-d²·3)` の
  ガウスプールでワープ量を深め、脈を terracotta に温める。離すと damp で復元（ステートレス）。
- `<Canvas flat>`＋sRGBパレットを `pow(2.2)` で線形化して出力、troika Text 不使用でオフライン安全。

## トーン調整（余白・ミニマル化）
初回は INK 支配で「黒い油膜」になった（erosion-fBm の値が小さく `v` が低く出て INK に寄る＋
emboss の乗算でさらに暗転）。`v = f·2.1 + 0.70` に持ち上げ PAPER 支配へ、深いINK混合を弱め、
emboss を暗転でなく raking sheen に。さらに基本周波数を 1.15→0.82 に下げて脈を大きく開き、
グレイン・スパークルを削って monaka の 余白・洗練 に寄せた。

## 罠 / 学び
- シンプレックス微分の符号：`mᵢ = 0.5−|xᵢ|²` の `dmᵢ/dv = −2xᵢ` を落とすと勾配が崩れる。
  導関数 fBm でも「回転したドメインの勾配」を `total`（累積回転行列）で戻して蓄積する必要がある。
- ビルド成功≠描画成功。headless_shell 実描画で「暗すぎ／脈の粒度」を2回見て調整して確定。
