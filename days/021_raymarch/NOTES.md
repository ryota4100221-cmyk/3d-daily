# Day 021 — Week 3 · CAPSTONE "Cairn" (レイマーチ / raymarched SDF)

**難易度 ★★★★**

## 狙い
Week 3 の弧（フルスクリーンfield 015 → 頂点変位 016 → ノイズ大理石 017 →
フレネルガラス 018 → ディゾルブ 019 → FBO流体 020）を、それらを**包摂する
1つの技法**で締めくくる週次まとめ。ジオメトリもライトも一切使わず、画面全体を
1枚のフラグメントシェーダで描く「静物」を作る。monaka のトンマナ（磁器の
ペーパーホワイト＋テラコッタ1点＋余白）。

## 新しく入れた技法（前日までの自分から1段）
- **レイマーチング（SDF スフィアトレーシング）** ＝ このプロジェクト初。
  three.js のジオメトリ／ライトはゼロ。フルスクリーンクアッド（Day015系譜）から
  1ピクセル1レイを飛ばし、符号付き距離場を `t += map(pos)` で前進させて面に当てる。
  「オブジェクト」は純粋な数式（sphere / roundBox / torus）。
- **smooth-minimum (smin) で溶接** — 交差でなく `smin(a,b,k)` で磁器の塊が
  互いに溶け合い1つの連続体（cairn＝積み石）になる。テラコッタ球も body へ
  smin で埋め込み「後付け」でなく「入れ子」に見せる。
- **SDF 勾配から法線を再構築（tetrahedron 4-tap）** — Day016 の「変位後の法線を
  解析的に作り直す」発想を、高さ場でなく体積（SDF）へ拡張。
- **iq ソフトシャドウ＋AO** — 同じ距離場を追加で数タップ・マーチしてペナンブラと
  接地の暗まりを得る（Day004 が蓄積バッファを要した効果を距離タップだけで）。
- **ドメインワープ fBm（Day017 系譜）で法線を微揺らし** — 磁器表面を無釉の
  セラミック微起伏に。ノイズを「色」でなく「素材」として使う。
- **フレネルリム（Day018）** `pow(1+dot(rd,N))` で、視線から背を向ける稜線に
  テラコッタの細い縁光り。
- **ポインタ＝ライト（Day018/015 系譜）** — damp したカーソルでキーライト方向を
  操作＋地面にテラコッタの温色プールを落とす。離すと基準へ復元（ステートレス）。

## 構図・仕上げ
- 手組みカメラ（`ro/ta` から `uu,vv,ww` 基底）。緩い自動オービット＋カーソル視差。
- 背景はほぼフラットなペーパー void（余白）。地平線にごく淡い暖色。
- キー＋半球フィル＋地面バウンス＋フラット環境光の床でシャドウ側が濁らないよう調整。
- `<Canvas flat>`＋`pow(2.2)` 線形化。troika Text 不使用でオフライン安全。

## 罠 / 学び
- 起動時セッションが **detached HEAD** 状態で、Day017〜020 のコミットが
  ローカル `main` に載っていなかった（origin/main には push 済みだった）。
  `git merge --ff-only` で main を 020 tip へ早送りしてから 021 に着手。
- 初版はフレネルのテラコッタが強すぎて磁器の陰側が「汚れた暗い面」に見えた
  → リム係数を 0.9→0.55・指数 3.2→4.0 に絞り、半球フィルを 0.60→0.78＋
  フラット環境光 0.10 を足して陰側を持ち上げたら磁器が清潔に読めた。
- テラコッタ球を前方（z=0.82）へ出してキーライトに当たる位置に置くと、
  「暗いこぶ」から「発光する差し色」へ変わった＝差し色は光の当たる場所に置く。
- headless swiftshader 実描画11sで、ReadPixels stall 警告のみ・JS/GLエラーなしを確認。
