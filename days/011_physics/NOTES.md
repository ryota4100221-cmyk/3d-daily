# Day 011 — Physics · Repose（R3F）

Week II「動きと対話」の物理回。カリキュラム #11（@react-three/rapier / 落下・衝突・積み上げ）。
テーマは "Repose" —— **終わらない静物**。紙の浅い皿に、丸みのあるフォームが1つずつ降ってきて、
転がり、ぶつかり、**本物のコリジョンで積み上がる**。積まれたピースは一息ついたのち静かにフェードして
プールへ戻り、ケルン（石積み）はほぼ一定の姿を保ちながら永遠に組み変わり続ける。皿をクリックすると
近傍のピースへ近接度に応じたインパルスが走り、山がふわりと崩れて積み直る。

## 狙い
- 「積んで崩す一発ネタ」ではなく、**恒常状態のインスタレーション**にする。前日までの instanced field / damped pointer の
  作法を、初めて “ソルバに解かせる力” へ接続する。
- monaka のトンマナ（余白・骨色ベース・テラコッタ差し色・英字エディトリアル）を物理でも崩さない。積み過ぎず、
  皿の中に収まる控えめなケルンに調律（LIVE=22）。

## 使った技法（今日の新規＝物理）
1. **`<Physics>` + `RigidBody`（Rapier）** — WASM は base64 インライン版（compat）なので**実行時フェッチ無し＝オフライン安全**。
   dynamic body が fixed の皿床＋リング状の fixed 壁へ落ちて衝突・スタック。
2. **命令的リサイクリング・プール** — N=40 個の body を1つの useFrame から Rapier API
   （`setTranslation` / `setLinvel` / `setGravityScale` / `sleep` / `applyImpulse`）で駆動。
   最古の park を spawn、容量到達で最古の live を retire（フェード→park）し、シミュレーションを**有界かつ無限**に保つ。
   コライダは固定サイズのまま、**見た目メッシュの scale だけ damp** でイン／アウトしてテレポートを隠す。
3. **ポインタ → インパルス** — Day 009 の raycast→平面を流用。クリックのワールド座標を種に
   `exp(-d²/2σ²)` のガウス重みで resting body を突き上げる（変換ではなく力）。
4. 補助: `ccd` と厚い床スラブ＋固定 timeStep=1/60 で**トンネリング対策**（可変 timeStep は低FPS時に床を貫通したため撤去）。

## 詰まった所 / 学び
- 最初 `timeStep="vary"` にしたら、ヘッドレス（低FPS）で1ステップが巨大化し body が床を**すり抜けて y=-46, -169 へ落下**。
  → 固定 timeStep=1/60 ＋ 床を厚いスラブ（half=3）＋ dynamic に `ccd`。加えて**スポーンをフレーム蓄積ではなく
  ウォールクロック（`elapsedTime`）でスケジュール**して、フレーム落ちで一気に数十個湧く事故を防止。
- 白画面対策の系譜（Day003/007/008）を踏襲：HDRI フェッチせず手続き IBL、troika Text 不使用（タイポは DOM オーバーレイ）。
